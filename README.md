# prompt-diff

A lightweight, dependency-free CLI for side-by-side diffs of LLM prompt outputs.

## Installation

```bash
uv pip install .
```

## Usage

```bash
prompt-diff [options] <file_a> <file_b>
```

Compare two files containing LLM prompt outputs:

```bash
prompt-diff response_a.txt response_b.txt
```
